## Log into Rivanna

Ssh into the system from your terminal. Windows users use PuTTY.

```
ssh -Y your_UVA_ID@rivanna.hpc.virginia.edu
```
  
Check out your acccount and allocation status:

```
>whoami
```

```
>allocations
```

Where am I?

```
>pwd
```
